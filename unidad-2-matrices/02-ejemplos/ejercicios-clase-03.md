# Matrices — Día 3: patrones y secuencias

Hoy el desafío es distinto: se mostrará una matriz generada y se deberá **descifrar la regla** que produce esos valores para luego construir el código desde cero.

No se trata de memorizar fórmulas. Se trata de observar, encontrar el patrón y traducirlo a código usando solo lo que ya se conoce: doble `for`, índices `i` y `j`, y operaciones básicas.

---

## 🎯 ¿Qué se trabajará hoy?

- Leer una matriz y deducir la regla que genera sus valores.
- Traducir un patrón visual a una expresión con `i` y `j`.
- Construir matrices con patrones numéricos, posicionales, diagonales y mixtos.
