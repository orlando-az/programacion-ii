# 💻 Programación II

Repositorio oficial de apoyo para la asignatura **Programación II**, orientado al desarrollo de habilidades en el manejo de estructuras de datos y la construcción de soluciones informáticas aplicadas.

---

## 🎯 Objetivo de la asignatura

Desarrollar programas computacionales que permitan dar solución a problemas del entorno mediante el uso de estructuras de datos como:

- Arreglos (vectores y matrices)
- Registros
- Archivos

---

## 📚 Contenidos del curso

- Unidad 1: Arreglos
- Unidad 2: Matrices
- Unidad 3: Registros
- Unidad 4: Manejo de archivos

---

## 🧠 Metodología de trabajo

El aprendizaje se desarrollará bajo el enfoque:

**Concepto → Ejemplo → Práctica**

- Explicación breve de conceptos
- Desarrollo de ejemplos en clase
- Resolución de ejercicios progresivos
- Aplicación en un proyecto final

## 🚀 ¿Qué es GitHub?

GitHub es una plataforma que permite almacenar, organizar y compartir código de programación. Funciona como un repositorio donde puedes:

- Guardar tus proyectos
- Ver ejemplos de código
- Descargar ejercicios
- Subir tus propias soluciones

En esta materia, GitHub será utilizado como herramienta de apoyo para el aprendizaje y seguimiento del curso.

- Revisar el repositorio después de cada clase
- Resolver los ejercicios propuestos
- Practicar de forma constante
- Subir avances del proyecto cuando se indique
