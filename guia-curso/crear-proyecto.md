# 🚀 Guía Básica: Ejecutar un Proyecto en C# usando VS Code

## 🎯 Objetivo

Aprender a crear, ejecutar y probar un proyecto en C# utilizando Visual Studio Code paso a paso.

---

## 🧰 Requisitos Previos

Antes de comenzar, asegúrate de tener instalado:

- Visual Studio Code
- .NET SDK
- Extensión de C# en VS Code

---

## 📁 1. Crear o abrir una carpeta de trabajo

1. Abrir VS Code
2. Ir a **Archivo → Abrir carpeta**
3. Crear o seleccionar una carpeta (ej: `MiProyectoCSharp`)

---

## ⚙️ 2. Crear un proyecto en C#

Abrir la terminal en VS Code:

- Atajo: `Ctrl + ñ` o `Ctrl + ``

Ejecutar el siguiente comando:

```bash
dotnet new console
```
